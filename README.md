#  Maarg – Ride Booking Platform

##  Overview

Maarg is a full-stack ride-booking application that connects users with drivers (captains) through a structured ride lifecycle system.

This repository is divided into:

* **Frontend** – User Interface (React)
* **Backend** – API & Business Logic (Node.js + Express)

---

##  Project Structure

```
MAARG/
│── BACKEND/     # Server-side logic (API, DB, Auth)
│── FRONTEND/    # Client-side UI (React)
```

---

##  Quick Start

### Run Backend

```bash
cd BACKEND
npm install
npm start
```

### Run Frontend

```bash
cd FRONTEND
npm install
npm run dev
```

---

##  Features

* Ride booking system
* User & Captain modules
* Ride confirmation workflow
* Clean modular architecture
